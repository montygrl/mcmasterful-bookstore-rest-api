# McMasterful Bookstore — REST API

A TypeScript REST API for the McMasterful bookstore, built with a clean 
adapter pattern separating business logic from HTTP routing. Includes 
price filter fix converting string query parameters to numbers.

## Tech Stack

- **Language:** TypeScript (80%) / JavaScript (20%)
- **Environment:** VS Code Dev Container
- **CI:** GitHub Actions

## Project Structure

- `src/` — API route handlers and business logic
- `adapter/` — Data access and transformation layer

## Getting Started

Open this repository as a development container in VS Code.

```bash
npm install
npm run dev
```
